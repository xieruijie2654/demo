字体

书呆子字体#

Oh My Posh 旨在使用Nerd Fonts。Nerd Fonts 是流行的字体，经过修补以包含图标。我们推荐Meslo LGM NF，但任何书呆子字体都应该与标准主题兼容。

要看到的图标显示在哦，我的辣妹，安装一个书呆子字体，并配置您的终端使用它。

窗户#

下载您选择的 Nerd 字体，并在系统范围内安装该字体。有关更多上下文，请参阅此线程。

视窗终端#

安装 Nerd 字体后，您需要配置 Windows 终端才能使用它。这可以通过修改 Windows 终端设置轻松完成（默认快捷方式：）CTRL + SHIFT + ,。在您的settings.json文件中，在fontFace属性下添加 defaults属性profiles：

{
“个人资料” ：
{
“默认” ：
{
"fontFace" : "MesloLGM NF"
}
}
}

其他字体#

如果您对使用书呆子字体不感兴趣，您将希望使用不包含任何书呆子字体图标的主题。该minimal主题不使用书呆子字体图标。

创建自己的主题也始终是一种选择😊
